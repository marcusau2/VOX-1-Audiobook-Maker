==========================================
VOX-1 PORTABLE VERSION SETUP INSTRUCTIONS
==========================================

EASY 3-STEP SETUP (5-10 minutes total)
---------------------------------------

STEP 1: Download Python
------------------------
Run: Download-Python.bat

This will automatically download Python 3.10.11 embedded version (~9 MB)
No installation or admin rights required!


STEP 2: Run Setup
------------------
Run: Setup-Portable.bat

This will automatically:
- Configure Python environment
- Download VOX-1 application from GitHub
- Install all dependencies (~2GB, includes PyTorch)
- Set up the launcher

NOTE: You need internet connection for this step!
The download may take 5-10 minutes depending on your connection.


STEP 3: Launch VOX-1
--------------------
Double-click: Launch VOX-1 Portable.vbs

The app will open in your browser!
No Python installation needed, no admin rights required.

==========================================
TROUBLESHOOTING
==========================================

"Python 3.10 not found!"
-------------------------
Run Download-Python.bat first.
Verify python310/ folder exists with python.exe inside.


"Failed to download from GitHub!"
----------------------------------
Check your internet connection.
If repository is private and you don't have access:
- Ask the repository owner to make it public temporarily, OR
- Get a direct download link from the owner


"Some packages failed to install"
----------------------------------
This is usually okay. Try:
1. Close and rerun Setup-Portable.bat
2. Check internet connection
3. If it persists, the app may still work - just try launching it!


"App won't start"
-----------------
1. Make sure you ran both Download-Python.bat AND Setup-Portable.bat
2. Check that app/ folder exists and has files inside
3. Right-click Launch VOX-1 Portable.vbs → Open to see error messages


==========================================
SYSTEM REQUIREMENTS
==========================================

- Windows 10/11 64-bit
- NVIDIA GPU with 8GB+ VRAM (12GB recommended)
- Internet connection (for initial setup only)
- ~5GB free disk space


==========================================
MOVING TO ANOTHER PC
==========================================

After setup is complete, you can:
1. ZIP the entire VOX-1-Portable/ folder
2. Copy to any Windows PC with compatible GPU
3. Extract and run Launch VOX-1 Portable.vbs
4. No additional setup required!

Note: The new PC must have:
- NVIDIA GPU with appropriate drivers
- Windows 10/11
